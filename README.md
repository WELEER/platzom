# Platzom

Es un programa de ejemplo de como utilizar NPM, Hyper,
node.js inpartido por [platzi](https://platzi.com) en fundamentos de javascript

## Descripcion del programa

- si la palabra termina con "ar" se le quita esas dos letras
- si la palabra incia con z se le añade "pe" al final
- si la palabra traducida tiene 10 o mas letras, se debe partir a la mitad y unir con un guion medio
- si las palabra original es un palindromo, ninguna regla cumple, se intercalan entre mayusculas y minusculas

## Instalacion

```
npm install platzom

```

# Uso

```
import platzom from 'platzom'


platzom('abcdcba')

```
